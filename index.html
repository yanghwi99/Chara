자주 묻는 질문

궁금한 점이 있으시면 아래에서 확인해보세요.

용량은 어느정도까지 가능한가요?

용량상관없이 긴 영상파일이라도 한번에 올릴 수 있어요.

분석된 데이터는 어디에 저장되나요?

영상 파일은 외부 서버에 업로드되지 않고 모든 처리가 내 PC에서 이루어집니다. 분석 결과는 로컬에 저장되며 Chara는 영상 원본 데이터를 수집하거나 보관하지 않아요.

채팅 데이터는 어떻게 가져오나요?

VOD URL을 입력하면 채팅 데이터를 자동으로 불러옵니다. 직접 채팅 파일을 가져오실필요 없어요.

영상 처리 시간은 얼마나 걸리나요?

8시간짜리 영상 기준 평균 10~15분 내외입니다. 처리 중에는 다른 작업을 하셔도 되고, 완료되면 알림을 보내드려요.
