Chara 사용방법

처음 사용하시는 분도 쉽게 따라할 수 있도록
주요 기능별로 정리했습니다.

01

프로그램 다운받기

이메일에서 발송드린 링크를 타고 들어가 설치합니다.

02

회원가입

프로그램을 실행하면 로그인/회원가입 화면이 나타납니다.

03

프로젝트 생성

  1. 좌측 상단 새 프로젝트 생성 버튼 클릭
  2. 영상 선택 — 편집할 영상 파일을 불러옵니다.
  3. 생방송 다시보기 URL 입력 — 채팅을 불러옵니다.
  4. 포맷 선택
    · 롱폼: 20분 정도의 스토리 있는 내용으로 편집카드 생성
    · 숏폼: 1분 이내 재밌는 내용으로 편집카드 생성
  5. 편집 의도 선택
    · 흐름살리기: AI가 큰 맥락에서 편집할 부분을 알려줍니다
    · 하이라이트 추출: AI가 1분 이내의 하이라이트 부분을 알려줍니다
  6. 편집카드 장수 선택 후 분석 시작
편집할 영상 파일과 생방송 다시보기 URL의 영상 길이가 동일해야 합니다.
예) 편집할 영상 4시간 + 다시보기 URL 4시간 → O
예) 편집할 영상 4시간 + 다시보기 URL 12시간 → X
04

쇼츠 만들기

원하시는 기능을 선택하시면 AI가 쇼츠를 생성해줍니다.

이런 식으로 10개의 쇼츠를 만들어주고 이 중 5개를 선택하셔서 내보내시면 됩니다.

완료 후 들어오시면 쇼츠를 편집할 수 있는 영상이 생성되며, 원하시는 편집 스타일로 편집을 진행하시면 됩니다.

05

침묵 제거

영상에서 빈 음성(무음 구간)을 자동으로 제거해줍니다.

영상이 길 경우 토큰을 많이 사용하므로 주의해주시길 바랍니다.
06

Export

Export는 현재 영상을 저장합니다.

07

구간별 저장

자른 구간을 본인의 컴퓨터로 저장할 수 있습니다.

08

그 외 기능

영상 자르기
좌측 상단의 가위 모양을 클릭 → 시작점과 끝점을 표시
혹은 챗봇에게 "분할해줘"라고 말씀하시면 됩니다.
예) "8:12:20 - 8:15:50분 분할해줘"
원본 영상으로 돌아가기
챗봇에게 "원본영상으로 돌아가줘" 입력
